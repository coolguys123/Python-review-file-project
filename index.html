Problem Statement

Create a Python program to perform the following tasks:

  1. Create a function called fileWrite() to write 77 random numbers between 1 and 7777 into a file named "First_Last_Num.txt", with each number on a new line.
  2. Create a function called fileRead() to read the 77 numbers from the file and handle any file reading errors using exception handling.
  3. Create additional functions:
    • printNum(): Display the numbers in a format of 7 numbers per row (10 rows in total).
    • sum(): Return the sum of the 77 numbers.
    • writeAverage(): Calculate and write the average of the numbers to the file, formatted to 2 decimal places.
  4. Prompt the user to repeat the process for another set of 77 numbers.
  5. Create a main() function to orchestrate these tasks, and call it to execute the program.

Solution Guide

  1. Function: fileWrite() 
    import random

def fileWrite():
    with open("First_Last_Num.txt", "w") as file:
        for _ in range(77):
            file.write(f"{random.randint(1, 7777)}\n")
  2. Function: fileRead() (with error handling) 
    def fileRead():
    try:
        with open("First_Last_Num.txt", "r") as file:
            numbers = [int(line.strip()) for line in file]
        return numbers
    except FileNotFoundError:
        print("Error: File not found.")
        return []
  3. Function: printNum() 
    def printNum(numbers):
    for i in range(0, len(numbers), 7):
        print(" ".join(f"{num:5}" for num in numbers[i:i+7]))
  4. Function: sum() 
    def sum(numbers):
    return sum(numbers)
  5. Function: writeAverage() 
    def writeAverage(numbers):
    average = sum(numbers) / len(numbers)
    with open("First_Last_Num.txt", "a") as file:
        file.write(f"Average: {average:.2f}\n")
